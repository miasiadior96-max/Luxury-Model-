# Luxury Model Portfolio

A minimal, image-forward model portfolio site built with:

- Next.js App Router
- TypeScript
- Tailwind CSS

## Features

- Responsive layout
- Clean, editorial design
- Inline sections: Portfolio, About, Contact
- Easy to update and expand

## Deployment

Deploy instantly on [Vercel](https://vercel.com).

## Customize

- Edit content in `app/page.tsx`
- Replace placeholder images with real work
- Upload your `compcard.pdf` to `public/` when ready
